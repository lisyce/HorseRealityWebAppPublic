About This Project

I started this project to help out a friend who got tired of entering her horses' stats into an Excel spreadsheet! At first I was just going to hand her a script for Excel, but it evolved into this web app. I hope to keep expanding it in the future. :)

FAQ

Why does it take so long to load my horses?

When you request to get your horse data, we have to send several requests to the HorseReality servers per horse! Because we don't want to overwhelm the servers, we can only request so much data per second and so it can take quite some time to load it all.
